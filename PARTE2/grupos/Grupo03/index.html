Scrollspy - Grupo 03

Explicación profunda del componente Scrollspy (Bootstrap 5)

Descripción general

El componente Scrollspy de Bootstrap es una herramienta dinámica que sincroniza el desplazamiento del contenido (scroll) con los elementos de navegación de una página web. Su función principal es resaltar automáticamente el enlace activo en el menú según la sección visible del contenido, brindando al usuario una referencia clara sobre en qué parte del sitio se encuentra.

Funcionamiento general

Cuando el usuario se desplaza por una página extensa, Scrollspy "espía" el movimiento del desplazamiento y detecta qué sección está visible en ese momento. Bootstrap utiliza JavaScript interno para aplicar automáticamente la clase .active al enlace del menú correspondiente y eliminarla de los demás, generando una experiencia de navegación fluida y visualmente atractiva.

Este proceso ocurre en tiempo real mientras el usuario navega hacia arriba o hacia abajo, comparando la posición del scroll con las etiquetas id que definen cada sección. De esta forma, el sistema identifica la parte visible del documento y actualiza el estado del menú de navegación en consecuencia.

Estructura básica y atributos principales

Para implementar Scrollspy correctamente se deben configurar ciertos atributos HTML que permiten vincular el área con desplazamiento y el menú correspondiente:

  • data-bs-spy="scroll": activa el comportamiento Scrollspy.
  • data-bs-target="#id-del-menu": indica el menú que se actualizará automáticamente.
  • data-bs-offset="valor": establece el margen superior (en píxeles) para activar un enlace.
  • tabindex="0": permite que el contenedor reciba eventos de desplazamiento.

2) Ejemplo tomado del video

Explicación del código principal (capturas del video)

1) Activación del Scrollspy: En la etiqueta <body> se utilizan los atributos data-spy="scroll", data-target="#scrollspy" y data-offset="1" para vincular el desplazamiento de la página con la barra de navegación.

2) Barra de navegación: Se crea una barra fija con <nav id="scrollspy">. El id es importante porque será el objetivo que Scrollspy actualizará para resaltar el enlace activo.

3) Enlaces del menú: Los enlaces de navegación usan <a href="#seccion">. Scrollspy se encarga de agregar la clase .active automáticamente al enlace que coincide con la sección visible.

4) Secciones del contenido: Cada sección debe tener un id correspondiente a los enlaces del menú. Al desplazarse la página, Scrollspy detecta qué sección está en pantalla y resalta el enlace asociado.

4) Ejercicio de la Prueba

Ejemplo con NAVBAR — Explicación del código
1) Activación del ScrollSpy

En este ejemplo el espía se activa sobre el <body> porque usamos Bootstrap 3: data-spy="scroll", data-target="#MenuBar" y data-offset="80..100". Así, el menú superior se actualiza según la sección visible.

2) Barra de navegación fija

El menú es un <nav id="MenuBar" class="navbar navbar-inverse"> con position:fixed; top:0; width:100% para mantenerlo visible. El id #MenuBar es el mismo que se usa en data-target.

3) Enlaces del menú

Cada <a> apunta con href="#id" a su sección (#inicio, #docente, #comandante, #grupos). Scrollspy aplica .active al <li> correspondiente.

4) Secciones observadas

Las secciones del contenido usan id que coincide con los enlaces. Se agrega margin-top o padding-top (y un offset de ~100 px) para que el título no quede oculto bajo el navbar fijo.

5) Inicialización con jQuery (Bootstrap 3)

Se refuerza la activación con jQuery: $('body').scrollspy({ target: '#MenuBar', offset: 100 }); El offset debe aproximar la altura de tu navbar + espacio superior.
